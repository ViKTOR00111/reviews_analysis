### Пояснения по анализу отзывов
Кол-во отзывов слишком большое для быстрого анализа через бесплатный ИИ (около 22 тысяч суммарно по трём артикулам). 
Выяснена доля пустых отзывов (без текста, только с тегами) ~61%. Даже исключив их получаем около 13,5 тысяч отзывов по 3 товарам. 
Слишком много для контекста бесплатных ИИшек и без API-автоматизации этой ИИшки. Поэтому использовалась стратифицированная случайная 
выборка (5 страт, задаются кол-вом "звезд" в отзыве; страты пропорциональны) для каждого товара. 
Каждый товар - отдельный csv-файл и отдельный запрос в модель. Выборочный метод использовался не с каждым товаром, 
один из артикулов (790254015) имеет не такое большое кол-во отзывов, обрабатывались все отзывы с текстом. Использовался ChatGPT (бесплатная версия)

Понимаю, что логи в открытый репозиторий пушить не стоит, но они не содержат чувствительной информации и могут быть полезны как пример выполнения.
